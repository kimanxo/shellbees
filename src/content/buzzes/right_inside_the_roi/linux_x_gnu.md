## Key Concepts

- **Kernel:** The core component of Linux that manages hardware and system resources.

- **GNU:** A collection of free software tools that, together with the Linux kernel, form a complete operating system.

- **Shell:** The command-line interface used to interact with the operating system.

- **Distribution (Distro):** A complete Linux operating system built around the Linux kernel, such as Ubuntu, Debian, Fedora, or Arch Linux.

- **Package Manager:** A tool used to install, update, and remove software (e.g., APT, DNF, Pacman).

- **File System:** The hierarchical structure Linux uses to organize files and directories.
